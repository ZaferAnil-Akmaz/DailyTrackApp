# Beslenme ve Antrenman Takibi

Tarayıcıda çalışan, Local Storage tabanlı kişisel beslenme, supplement ve antrenman takip uygulaması.

## GitHub Pages ile yayınlama

1. Bu klasörü GitHub'da yeni bir repoya gönderin.
2. Varsayılan dalın `main` olduğundan emin olun.
3. GitHub'da `Settings > Pages` bölümünden kaynak olarak `GitHub Actions` seçin.
4. `Deploy GitHub Pages` iş akışı tamamlandığında uygulama repo Pages adresinde açılır.

Uygulama verileri her cihazın kendi Local Storage alanında tutulur; GitHub Pages'e veri gönderilmez.
